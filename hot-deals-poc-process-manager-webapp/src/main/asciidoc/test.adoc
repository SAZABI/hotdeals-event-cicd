[[result-code]]
== 응답 코드
API 호출의 result, result_msg 응답값

[[result-code-success]]
=== 성공
[cols="2,8"]
|===
| 응답코드 | 메시지

| 200
| 성공

| 201
| 생성 성공

| 202
| 처리 중

| 204
| 데이터가 없습니다.
|===

[[result-code-client-error]]
=== 클라이언트 에러
[cols="2,8"]
|===
| 응답코드 | 메시지

| 400
| 요청한 페이지를 찾을 수 없습니다.

| 401
| 로그인 후 다시 시도해주세요.

| 402
| 결제 후 다시 시도해 주세요.

| 403
| 접근 권한이 없습니다.

| 404
| 죄송합니다. 해당 페이지를 찾을 수 없습니다.

| 405
| 요청 내용에 명시된 메소드를 수행하기 위해 해당 자원의 이용이 허용되지 않습니다.

| 406
| 지원하지 않는 미디어 타입 입니다.

| 409
| 리소스 상태에 의하여 요청을 수행할 수 없습니다.

| 412
| 파라미터를 확인 후 다시 시도해주세요.
|===

[[result-code-server-error]]
=== 서버 에러
[cols="2,8"]
|===
| 응답코드 | 메시지

| 500
| 시스템 에러가 발생했습니다. 잠시 후 다시 시도해주세요.

| 503
| 서버에 부하가 많아서 잠시 접근이 거부 되었습니다.

| 504
| 게이트웨이나 프록시 타임아웃이 발생했습니다. 잠시 후 다시 시도 해 주세요.
|===
